¿Qué diferencias y semejanzas hay entre Java y JavaScript? Diferencias:
- Java es un lenguaje compilado, mientras que JavaScript es interpretado.
- Java se usa principalmente para aplicaciones de escritorio y móviles, mientras que JavaScript se usa en desarrollo web.
- Java requiere un entorno de ejecución (JVM), mientras que JavaScript se ejecuta en navegadores.
Semejanzas:
- Ambos tienen una sintaxis similar en ciertos aspectos.
- Ambos pueden usarse en desarrollo web (Java en backend, JavaScript en frontend y backend con Node.js).

¿Cómo se declara una variable con alcance local dentro de una función? Se usa let o const dentro de la función, por ejemplo:
function ejemplo() { let x = 10; }
¿Qué implicaciones tiene utilizar variables globales dentro de funciones? - Puede provocar efectos secundarios si la variable es modificada accidentalmente.
- Dificulta la depuración y mantenimiento del código.
- Puede generar conflictos de nombres en programas grandes.
- Reduce la modularidad y reutilización del código.
